Проект с разбором небольшого фрагмента JavaScript-кода, где используется `Date.now()`, немедленно вызываемая функция и измерение времени выполнения через `console.time`.

Что было разобрано:
- переменная `x` появляется в момент выполнения выражения `x = Date.now()`;
- в `x` записывается текущее время в миллисекундах на момент запуска программы;
- точное значение `x` заранее определить нельзя, потому что оно зависит от времени запуска;
- переменная `x` нужна для фиксации начала отсчета и проверки, сколько времени прошло;
- IIFE - это функция, которая создается и сразу вызывается;
- в этом примере IIFE используется только для немедленного выполнения кода.

Без IIFE это выражение можно переписать так:

```js
console.time('a');
const y = 2 ** 15;
const x = Date.now();
while (Date.now() - x < y);
console.timeEnd('a');
```

Этот вариант работает так же, как и исходный. Сначала запускается таймер, потом задаются значения `y` и `x`, после чего цикл `while` ждет, пока не пройдет нужное количество миллисекунд. В конце `console.timeEnd('a')` показывает примерно тот же результат, чуть больше 32 секунд.
