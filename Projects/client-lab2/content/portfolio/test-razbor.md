1. Переменная `x` появляется в момент выполнения выражения `x = Date.now()`. В неё записывается текущее время в миллисекундах на момент запуска программы. Точное значение заранее определить нельзя, потому что оно зависит от времени запуска. Эта переменная нужна для того, чтобы зафиксировать начало отсчёта и потом проверить, сколько времени прошло.

2. IIFE — это функция, которая создаётся и сразу вызывается. В данном случае она используется только для немедленного выполнения кода. Без IIFE это выражение можно переписать так:

```js
console.time('a');
const y = 2 ** 15;
const x = Date.now();
while (Date.now() - x < y);
console.timeEnd('a');
```

Этот вариант работает так же, как и исходный. Сначала запускается таймер, потом задаются значения `y` и `x`, после чего цикл `while` ждёт, пока не пройдёт нужное количество миллисекунд. В конце `console.timeEnd('a')` показывает примерно тот же результат более  32 секунд.
