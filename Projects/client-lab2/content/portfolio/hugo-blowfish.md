Персональный сайт-портфолио, созданный на статическом генераторе Hugo с использованием темы Blowfish. Проект включал настройку структуры сайта, конфигурации, навигации, страниц с контентом и публикации на GitHub Pages. В процессе работы я занимался исправлением путей, конфигурации baseURL, логики деплоя и организацией проекта в репозитории.

Что было сделано:
- настройка Hugo-проекта с нуля;
- подключение и конфигурация темы Blowfish;
- создание структуры страниц для портфолио;
- настройка деплоя на GitHub Pages;
- исправление ошибок публикации, путей и отображения стилей;
- подготовка проекта к дальнейшему наполнению контентом.
